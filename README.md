# Python Design Patterns Recipes

A concise collection of Python recipes inspired by the Gang of Four (GoF) design patterns.

## Purpose

- Learn classic object-oriented design patterns through small, readable Python examples.
- Focus on practical usage, tradeoffs, and when to apply each pattern.

## What You’ll Find

- Pattern-oriented recipe modules (creational, structural, behavioral).
- Minimal examples that highlight intent over framework complexity.
- Notes about Pythonic alternatives where relevant.

## Usage

Run any recipe module directly, for example:

```bash
python path/to/recipe.py
```

## Audience

Python developers studying software design and architecture fundamentals.
